# ChaosGetCrash
监听APP崩溃日志
对于上线的APP出现Crash监听问题出处，通过运用UncaughtExceptionHandler,我们可以把崩溃日志存储到手机内存或者上传到服务器。
这样我们在后台就可以定位崩溃来源，个人觉得是一个很实用的后期项目维护手段。
